MDU Debug Console

Live monitoring for the STM32 native USB CDC debug stream behind the USB2514 hub. The app mirrors the MCU's USB broadcast, decodes the per-board SDU telemetry frames (base 0x080 + 8*B + S for sensors 0-4) and falls back to SLCAN for other CAN IDs.
